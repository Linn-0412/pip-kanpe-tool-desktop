# 機能仕様

## 目的

PiP Kanpe Tool Desktopは、FF14などの攻略中に参照したいカンペ画像をローカルで管理し、常に手前表示の小窓で素早く切り替えるためのデスクトップアプリです。

ブラウザ版のDocument Picture-in-Pictureや拡張機能に依存せず、Tauriの小窓とグローバルショートカットで操作できることを主目的にします。

## 対象環境

- 主対象OS: Windows
- 配布形式: GitHub ReleasesのWindowsインストーラー
- FF14表示モード:
  - フルスクリーン: PiP小窓は表示不可
  - 仮想フルスクリーン: 表示可能
  - ウィンドウモード: 表示可能

## 画像登録

- 登録可能な画像は最大80枚です。
- 画像の追加方法:
  - ファイル選択
  - ドラッグ&ドロップ
  - `Ctrl+V` によるクリップボード画像貼り付け
- 複数ファイル選択時は、ファイル名の自然順で登録します。
  - 例: `P1_1.png`, `P1_2.png`, `P1_10.png`
- クリップボード画像は、元の名前が汎用名の場合に `clipboard-YYYYMMDDhhmmss-連番.ext` の名前を補います。
- 保存時の軽量化が有効な場合、画像は長辺2048pxに収まるよう圧縮します。
- 画像はサーバーへアップロードせず、ローカルのアプリデータに保存します。

## 登録画像の操作

- 登録画像はカード一覧として表示します。
- カードをクリックすると、その画像を現在の表示対象にします。
- `↑` / `↓` で並び順を変更できます。
- `×` で画像を削除できます。
- 目アイコンで、プレビュー/PiP表示対象から一時的に除外できます。
  - 非表示画像はリストには残ります。
  - もう一度押すと表示対象へ戻ります。
- 画像名は後から変更できます。
- 全削除ボタンで登録画像をまとめて削除できます。

## グループ

- グループを作成して画像を分類できます。
- グループ名は最大24文字です。
- 表示グループを選ぶと、そのグループに所属する画像だけを表示・切り替え対象にします。
- 選択中のグループで画像を追加した場合、そのグループへ自動で所属します。
- 登録済み画像はカード内のチェック操作でグループ所属を切り替えられます。
- グループ削除時、画像自体は削除せず、対象グループの所属だけを外します。

## カンペセット

- カンペセットは、登録画像、並び順、グループ、PiP表示設定をまとめて扱う機能です。
- エクスポート:
  - `.pipkanpe` ファイルとして保存します。
  - 画像本体はDataURLとして含めます。
- インポート:
  - `.pipkanpe` またはJSONファイルを読み込みます。
  - 現在の登録枚数と合わせて80枚を超える場合は読み込みません。
  - 同名グループが既にある場合は既存グループへ統合します。
  - 新しいグループ名は追加します。

## プレビュー

- メイン画面右側に現在画像のプレビューを表示します。
- 表示モード:
  - 全体を表示
  - 画面いっぱい
- プレビューの矢印操作で前後のカンペへ切り替えられます。
- キーボードの `ArrowLeft` / `ArrowRight` でも切り替えられます。
  - 入力欄、ボタン、セレクトなどにフォーカスがある場合は画面操作を優先します。

## PiP小窓

- `PiPで表示` ボタンで常に手前表示の小窓を開きます。
- 小窓には現在の画像、前/次ボタン、必要に応じて画像名と枚数を表示します。
- 小窓の前/次ボタンでカンペを切り替えられます。
- メイン画面で画像を切り替えると、小窓側にも反映します。
- 小窓のサイズ:
  - プリセット: `480x270`, `640x360`, `800x450`, `960x540`
  - カスタム: 幅320から1280、高さ180から720
- タイトルバー非表示:
  - 有効にするとOS標準のタイトルバーを消します。
  - 画像部分をドラッグして小窓を移動できます。
  - ホバー時に閉じるボタンを表示します。

## PiP表示設定

- 矢印と画像名の表示位置:
  - 上部
  - 中央
  - 下部
- 矢印と画像名の大きさ:
  - 小
  - 中
  - 大
- 矢印ボタンの配置:
  - 横並び
  - 左に縦並び
  - 右に縦並び
- 矢印の当たり判定を縦いっぱいにする:
  - 縦並び配置では、矢印と同じ縦ラインを小窓全体の操作領域として扱います。
  - ラインの上半分を前、下半分を次として扱います。
- 背景の濃さ:
  - 通常
  - 薄め
  - 透明
- コントロールパネルを画像に重ねない:
  - 有効時は画像領域と操作領域を分けます。
- マウス操作時だけ表示する:
  - 有効時は通常時に操作UIを隠し、ホバー時に表示します。
- 画像名と枚数を表示する:
  - 無効時はラベルを非表示にします。
- 画像名の拡張子を表示する:
  - 有効時は `phase.webp` のように拡張子まで表示します。

## グローバルショートカット

- デスクトップ版ではTauriのglobal-shortcut pluginを使って、ゲーム中でも前後切り替えできます。
- 初期値:
  - 前のカンペ: `Ctrl+F5`
  - 次のカンペ: `Ctrl+F6`
- 設定方法:
  - 入力欄へ直接入力
  - `キー入力` ボタンで押したキーを取り込む
- 前/次に同じショートカットは設定できません。
- OSや他アプリ、FF14側のキーバインドと衝突する場合は登録に失敗する、または期待通り動作しない可能性があります。

## 使い方ガイド

- 初回起動時に使い方ガイドを表示します。
- `次回以降表示しない` を有効にすると、次回起動から自動表示しません。
- `使い方` ボタンからいつでも再表示できます。

## 支援リンク

- `支援する` ボタンからOFUSEを開きます。
- URL: `https://ofuse.me/linn0412`
- Tauri環境では外部ブラウザで開きます。

## 更新確認

- `更新を確認` ボタンでGitHub Releases上の更新情報を確認します。
- 更新がある場合、ユーザー確認後にダウンロードとインストールへ進みます。
- 起動時にも自動で更新確認します。
