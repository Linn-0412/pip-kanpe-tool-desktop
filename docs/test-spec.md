# テスト仕様

## English Summary

CI should run before release builds and verify dependency installation, release metadata, frontend build, JavaScript syntax, core unit tests, Rust formatting, Rust type checks, Rust tests, and clippy warnings.

Manual release testing should cover app startup, image registration, image operations, groups, deck import/export, PiP subwindow behavior, global shortcuts, kanpe editor behavior, persistence after restart, update checks, support links, and language switching.

## CIで確認すること

GitHub Actionsでは、Releaseビルド前に次を確認します。

- 依存関係をクリーンインストールできること
- リリースメタデータが一致していること
- Webアセットをビルドできること
- JavaScriptの構文エラーがないこと
- `core.js` の純粋関数テストが通ること
- Rustのフォーマットが崩れていないこと
- Rustの型チェックが通ること
- Rustのテストが通ること
- Rustのclippy警告がないこと

## ローカル基本確認

リリース前に最低限確認します。

- `npm run release:check`
- `npm run check`
- `npm test`
- `npm run tauri:dev`
- 必要に応じて `npm run tauri:build`

## 手動確認チェックリスト

### 起動

- アプリが起動する
- 初回ガイドが表示される
- `次回以降表示しない` が保存される
- `使い方` ボタンでガイドを再表示できる

### 言語切替

- 初期表示で日本語または英語が選択される
- 言語セレクトで日本語から英語へ切り替えられる
- 言語セレクトで英語から日本語へ切り替えられる
- アプリを再起動しても選択した言語が残る
- PiP小窓のタイトル、前/次ボタン説明、空表示メッセージが選択言語に合う

### 画像登録

- ファイル選択で1枚登録できる
- ファイル選択で複数枚登録できる
- 複数枚登録時にファイル名順で並ぶ
- ドラッグ&ドロップで登録できる
- `Ctrl+V` でクリップボード画像を登録できる
- 80枚上限を超えて登録できない
- 画像軽量化ON/OFFで登録できる

### 画像操作

- カードクリックでプレビューが切り替わる
- `↑` / `↓` で並び替えできる
- 画像名を変更できる
- 目アイコンで表示/非表示を切り替えられる
- `×` で1枚削除できる
- 全削除できる

### グループ

- グループを追加できる
- グループ名を変更できる
- グループを削除できる
- 表示グループで絞り込める
- 選択中グループで画像を追加すると自動所属する
- 既存画像の所属を切り替えられる

### カンペセット

- `.pipkanpe` をエクスポートできる
- `.pipkanpe` をインポートできる
- グループとPiP表示設定が復元される
- 上限80枚を超えるインポートが拒否される

### PiP小窓

- `PiPで表示` で小窓が開く
- 小窓が常に手前に表示される
- メイン画面の現在画像が小窓に反映される
- メイン画面で切り替えると小窓も更新される
- 小窓の前/次ボタンで切り替えできる
- PiPサイズプリセットが反映される
- カスタムサイズが反映される
- タイトルバー非表示が反映される
- タイトルバー非表示時に画像部分ドラッグで移動できる
- ホバー時の閉じるボタンで閉じられる
- 縦並び+縦いっぱい当たり判定が小窓全体の縦ラインで動作する

### ショートカット

- 初期値 `Ctrl+F5` / `Ctrl+F6` で前後切り替えできる
- 入力欄へ直接入力して変更できる
- `キー入力` ボタンで変更できる
- 同じショートカットを前/次へ設定した場合にエラーになる
- FF14起動中、仮想フルスクリーンまたはウィンドウモードで切り替えできる

### カンペ作成

- カンペ作成画面を開ける
- 背景を切り替えられる
- 図形、マーカー、ギミック、ロール、ジョブアイコン、テキストを追加できる
- 素材をドラッグで移動できる
- 大きさ、横幅、縦幅、向きを変更できる
- Deleteキーで選択素材を削除できる
- 画像として登録できる
- 登録した作成画像をPiP表示できる

### 保存

- 画像、グループ、設定を変更する
- アプリを終了する
- アプリを再起動する
- 画像、グループ、設定が残っている

### 更新

- `更新を確認` ボタンで確認できる
- 最新の場合は最新表示になる
- 更新がある場合は確認ダイアログが出る
- インストール後、次回起動でバージョンが上がっている

### 支援リンク

- `支援する` ボタンでOFUSEが外部ブラウザに開く
