# カンペ作成仕様

## 目的

アプリ内で簡易的な攻略カンペ画像を作成し、そのまま登録画像へ追加できるようにします。

外部の画像編集ソフトを使わず、よく使うFF14攻略図の部品をドラッグ操作で配置できることを重視します。

## キャンバス

- SVGキャンバスを使用します。
- サイズ: 1024 x 1024
- 形状: 正方形
- 背景:
  - 黒背景
  - 白背景
  - 透明
  - 絶妖星乱舞 P1からP5のフィールド画像

## 操作

- 素材ボタンを押すとキャンバスへ追加します。
- 追加した素材はドラッグで移動できます。
- 選択中の素材は、スライダーで調整できます。
  - 大きさ
  - 横幅
  - 縦幅
  - 向き
- 横幅/縦幅は最大1000%まで拡大できます。
- 選択中の素材はDeleteキーで削除できます。
- `選択を削除` ボタンで選択中の素材を削除できます。
- `全クリア` でキャンバス上の素材をすべて削除できます。

## 出力

- `画像として登録` を押すと、現在のキャンバスをPNG画像として登録画像リストへ追加します。
- 追加後は通常の登録画像と同じように、並び替え、グループ所属、名前変更、PiP表示ができます。

## 素材追加方針

- FF14で一般的に使われる攻略表現を優先します。
- 色は攻略中に直感的に理解しやすい色へ寄せます。
- 正確なフィールドが必要な場合は、コードで再現するより画像アセットとして追加することを優先します。
- 公式・外部素材を追加する場合は、利用条件をREADMEまたは素材フォルダのREADMEに記録します。
